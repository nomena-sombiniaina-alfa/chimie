# Chimie

Une plateforme pédagogique pour comprendre la matière à travers ses langages fondamentaux : **les atomes, les liaisons, les réactions, les unités de mesure**.

## Pourquoi ?

La chimie souffre souvent d'un même problème dans l'enseignement : les concepts sont présentés isolément, sans qu'on perçoive comment l'un est né de l'échec ou de l'incomplétude du précédent. On apprend Bohr sans savoir ce qu'il corrige chez Rutherford ; on parle d'orbitales sans relier Schrödinger à l'incertitude de Heisenberg ; on récite la configuration électronique sans la rattacher aux couches dessinées. On distingue acide fort et acide faible sur le papier, sans jamais voir une seule molécule se dissocier.

Ce projet réunit tout cela autour d'objets visuels animés et de schémas interactifs.

## Ce qu'on y trouve

### Modèle atomique
**Sept représentations historiques synchronisées** (Dalton 1803 → Chadwick 1932), plus la vue de Lewis, la configuration électronique détaillée (avec exceptions de Klechkowski), les tendances périodiques, les isotopes naturels et les liaisons préférentielles. Quand on change d'élément ou de charge, toutes les vues se mettent à jour ensemble : un atome, plusieurs façons de le regarder. Le tableau périodique complet est filtrable par catégorie.

### Liaisons chimiques
**Huit types de liaisons** classés en trois familles : intramoléculaires (ionique, covalente, métallique), intermoléculaires (hydrogène, Van der Waals), et théories explicatives (Lewis, VSEPR, orbitales moléculaires). Chaque type s'accompagne d'une animation canvas dédiée pour visualiser le mécanisme à l'œuvre.

### Réactions chimiques
**Treize réactions** organisées en quatre familles : transfert d'électrons (corrosion, combustion, piles, respiration), transfert de protons (acides, bases, neutralisation), réarrangement de liaisons (substitution, addition, élimination), formation et rupture de solides (précipitation, dissolution, équilibre ionique). Chaque réaction est mise en scène dans un bécher avec pipette ou salière : les molécules tombent, se dissocient, s'éparpillent. Pour la famille acide-base, un sélecteur permet de basculer entre la version forte (dissociation totale) et la version faible (équilibre, dissociation partielle) avec une mini-théorie associée. Chaque animation comporte une légende explicite, un bilan en temps réel, et s'arrête au bout d'un nombre fini de cycles.

### Unités de mesure
**Le Système international d'unités** : les sept unités de base, les vingt-deux unités dérivées, et les hors-SI couramment acceptées. Pour chacune, on trouve sa définition actuelle (post-réforme 2019), une légende qui explique chaque symbole de la formule, la frise des redéfinitions historiques, les conversions vers les unités usuelles et les pièges fréquents : confusion masse / poids, kilogramme étalon abandonné en 2019, becquerel qui ne mesure pas la nocivité, et autres. Les sept constantes physiques fondamentales sont également listées.

## Pour qui ?

- **Lycéens et étudiants** qui veulent visualiser ce que les équations décrivent.
- **Enseignants** cherchant un support interactif pour illustrer la chronologie des modèles atomiques, les types de liaisons, ou le déroulement des grandes familles de réactions.
- **Curieux** de chimie, physique ou métrologie qui veulent un coup d'œil rapide et fiable.

## Source des données

Toutes les données scientifiques sont cross-vérifiées sur plusieurs sources de référence : Wikipédia (FR et EN), NIST Atomic Spectra Database, IUPAC, WebElements, Société Chimique de France, ScienceNotes. Les sources consultées sont accessibles directement dans l'application via le pied de page.
